Help

Required setup

  • Chrome extension: loaded unpacked from this repository's extension folder.
  • Native host: registered with install-native-host.ps1.
  • Node.js: required to run the native host, currently expected at C:\Program Files\nodejs\node.exe.
  • yt-dlp: default path is C:\yt-dlp\yt-dlp.exe.
  • ffmpeg: required for merging video and audio; configure its path in settings.

Simple workflow

  1. Open the course page and log in.
  2. Open the extension and click Start video capture.
  3. If the page has been open for a while, click Scan page to search iframe/video/resources again without reloading.
  4. Press Play on the course page.
  5. When variants appear, choose one, click Verify, check the real title/duration, and only then click Download.
  6. If downloading starts, watch the Downloading status. Stop it with the red ■ button.
  7. If direct download is not possible, the extension can start Tab recording. It records the tab with audio.
  8. Before recording, Video Course Capture highlights the detected player area or iframe. Confirm the correct frame so the recording follows the video area, not the whole page.
  9. After confirmation, scrolling is locked, a 5-second countdown is shown, and recording starts from the same viewport position.
  10. When Recording is active, click Stop or press Ctrl+Shift+Y.

How to read statuses

  • Idle: the extension is not doing anything.
  • Capture armed: waiting for Play and searching for a stream.
  • Downloading: yt-dlp is running.
  • Recording: tab recording is running.
  • Error: check the error text and native-host.log.
